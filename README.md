## Component Features
#### Messages
- awake()
- start()
- update()
- onEnable()
- onDisable()
- onDestroy()

#### Others
- coroutine
- change script execution order
- require components
- disallow multiple components
  
  All features have a similar or same behavior to [Unity Game Engine](https://docs.unity3d.com/ScriptReference/MonoBehaviour.html).
